# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-14

### Changed

- Renamed the distributable and container-facing product name to `ndlocr-lite-adapter`
- Updated documentation to describe the service as an Immich OCR adapter under the new product name
- Switched the OCR backend target from `ndlkotenocr-lite` to `ndlocr-lite`

### Added

- FastAPI-based Immich machine-learning compatible OCR adapter
- NDLOCR-Lite subprocess runner and Immich OCR response transformer
- Dockerfile, compose example, and GitHub Actions workflows
- README, Japanese README, NOTICE, and third-party license summary
